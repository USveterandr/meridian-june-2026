# Meridian — Launch Distribution Kit

A legitimate, no-spam plan to get investwithmeridian.com in front of buyers, investors, and DR agents — and to fill the first-100 founding-agent campaign. Everything here is manual-submit or you-paste; there is **no automated mass-submission**, because auto-posting to directories produces spammy backlinks that can trigger Google penalties and get you banned from the good platforms. Quality placement beats volume.

Work top to bottom: **Foundations → Business listings → Portals → Social → Agent outreach.** Track each row in `distribution-tracker.csv`.

---

## 0. One-time foundations (do these first)

These make everything else pay off. Most are free and take an afternoon.

1. **Google Search Console** — add the property `https://investwithmeridian.com`, verify (DNS or the HTML file — you already serve token files from `web/public/`), and submit `https://investwithmeridian.com/sitemap.xml`. Request indexing for `/`, `/agents`, `/pricing`, `/blog`.
2. **Bing Webmaster Tools** — you already ship `BingSiteAuth.xml`, so verification is one click. Import from Search Console, then submit the same sitemap. Bing also feeds DuckDuckGo and ChatGPT search.
3. **Confirm crawlability** — `robots.txt` allows crawling and references the sitemap; `/agents` is now in `sitemap.xml`. Re-run the `/on-page-seo` and `/technical-seo` skills on `/agents` and the homepage after the copy changes ship.
4. **Schema** — add `RealEstateAgent` / `Organization` JSON-LD to the homepage and `Offer` markup to `/agents`. Run `/generate-schema` on each.
5. **Analytics + UTMs** — put a UTM on every link you post (see the tracker) so you can see which channel actually drives agent claims and buyer signups.

## 1. Free business & map listings (high trust, evergreen)

These are the legit equivalent of "submit my site everywhere" — real, reputable directories that help local SEO instead of hurting it. One consistent NAP (Name, Address, Phone) across all of them.

| Listing | Why | Cost |
|---------|-----|------|
| **Google Business Profile** | Shows in Maps + local search; biggest single win for a DR business | Free |
| **Bing Places** | Maps/local on Bing ecosystem | Free |
| **Apple Business Connect** | Apple Maps presence | Free |
| **LinkedIn Company Page** | Credibility + agent recruiting | Free |
| **Facebook Business Page** | Required for Marketplace + FB/IG ads later | Free |
| **Instagram Business** | Primary visual channel for DR real estate | Free |
| **DR chambers / associations** — e.g. the local real-estate association (AEI) and your regional chamber of commerce | Trust signals + agent network | Varies |

> Consistency matters more than count. Use the exact same business name, phone, and description everywhere (EN + ES).

## 2. Real-estate portals & marketplaces (verify rules first)

Candidate channels where DR listings and agencies legitimately appear. **Before posting to any, check that platform's posting rules** — some allow free agency/listing profiles, some are paid, some prohibit third-party reposting (the same ToS caution from the brand review applies). Treat this as a shortlist to qualify, not a blast list.

- **Supercasas** — the dominant DR portal; claim an agency profile.
- **Corotos** — large DR classifieds (OLX network); free listings.
- **Encuentra24 (DR)** — regional classifieds with a real-estate vertical.
- **Facebook Marketplace + DR real-estate Groups** — where a lot of local demand actually is; post individual listings where the group allows it.
- **Lamudi / Point2 / international "Dominican Republic property" aggregators** — verify whether they accept feeds or free profiles.
- **DR1.com community + classifieds** — long-running expat/investor audience.

Approach: create the **agency/brand profile** on each qualifying portal and link back to Meridian. Only post individual property listings where you hold the listing and the platform permits it.
